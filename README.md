# ChefMenuExpo (MAST5112 POE)

## Summary
Mobile app built with **React Native (Expo) + TypeScript** for MAST5112.  
Allows a chef to add, view, edit, delete and filter menu items. Home screen shows the full menu and average prices per course.

## Features
- Add menu item (name, description, course, price)
- Edit and delete items
- Filter screen to show items by course
- Home screen displays average price per course
- Clean, component-based code structure using Context API
- No persistent storage (in-memory array as required)
